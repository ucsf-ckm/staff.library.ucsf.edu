Popular Links Updated

June 25, 2014 by Richard Trott

As some of you have noticed, the Popular Links on the web site have been slightly tweaked. (And, hey, maybe next time, I’ll post something before we make the change. Sorry about the lapse.)

Based on usage and search statistics, we removed Web of Science from the Popular Links and replaced it with Lexi-Comp.

Also, based on declining usage, we removed the link to reserve a group study room. We want to see if having fewer links and a bit more white space between the links will make it more likely that users take the time to read and use the links.

We will almost certainly have to make the study room reservation link more prominent again before the start of the fall quarter. It may end up elsewhere on the home page. Or it might find its way back into the Popular Links.

In the meantime, users can still quickly and easily reserve a room by following the link for “Additional Study Spaces” on the lower left of the home page.

As always, your input is very welcome. Email LibraryWeb with suggestions for how we can improve the site for our users.
